# Release Gates for TypeScript Skill

## Activation Rate
- Target: **≥ 90 %** activation across all models.

## Token Cost
- Target: **≤ 10 %** of baseline token usage.

## Run History
- Each optimizer run is logged in `run-history.md`.
- Include timestamp, model, activation %, token delta, and any regressions.

---

## How to Run
```bash
skill-optimizer run --skill typescript
```
